 Vrinda Sales Annual Report Dashboard

This project presents an interactive Sales Dashboard built to analyze the annual performance of Vrinda’s business. The dashboard provides insights into sales trends, customer demographics, regional performance, and order distribution across multiple channels.

 Key Features:
Sales vs Orders Analysis: Visualizes monthly sales performance and order trends using a combination of bar and line charts.
Customer Demographics: Analyzes purchasing behavior based on age group and gender, highlighting key customer segments.
Order Status Overview: Displays the percentage of delivered, cancelled, returned, and refunded orders to evaluate operational efficiency.
Top Performing States: Identifies the top 5 states contributing to overall sales.
Gender-based Sales Insights: Compares sales contribution between men and women.
Channel-wise Order Distribution: Shows how different platforms (Amazon, Flipkart, Myntra, etc.) contribute to total orders.
 Insights:
Women customers contribute the majority of sales.
Amazon is the leading sales channel, followed by Myntra and Flipkart.
Maharashtra and Karnataka are the top-performing states.
High delivery success rate (~92%) indicates strong logistics performance.
Sales show seasonal trends with peaks in early months.
🛠️ Tools & Technologies:
Microsoft Excel (Pivot Tables, Charts, Slicers)
Data Visualization Techniques
 Purpose:

The goal of this project is to demonstrate data analysis and visualization skills, helping businesses make data-driven decisions by identifying trends, patterns, and opportunities.
